# Car Sales Analysis

## Overview

This project analyzes car sales data using **Excel** and **Power BI** to identify sales trends, category-wise performance, and key business insights. The data was cleaned and prepared in Excel before building an interactive Power BI dashboard.

## Tools Used

- Microsoft Excel
- Power BI

## Project Steps

- Cleaned and prepared the dataset in Excel.
- Checked for missing values and inconsistencies.
- Performed basic Exploratory Data Analysis (EDA).
- Created KPIs to track sales performance.
- Built an interactive Power BI dashboard.
- Analyzed sales trends and category-wise performance.

## Key Insights

- Identified top-performing car categories.
- Analyzed monthly and yearly sales trends.
- Compared sales across different brands and models.
- Tracked overall sales performance using KPIs.
- Created interactive filters for detailed analysis.
